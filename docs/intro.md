# Orkestra Documentation

Welcome to the **Orkestra** docs. This is the starter page — replace it with
your own markdown. Any `.md` / `.mdx` file under `docs/` automatically becomes
a page, and folders become collapsible sidebar categories.

:::tip
Admonitions, code blocks with copy buttons, tables, and frontmatter all work
out of the box.
:::

```bash
cd website
npm install
npm run start
```
